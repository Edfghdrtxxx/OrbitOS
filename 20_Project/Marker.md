## Marker

[marker-pdf](https://github.com/VikParuchuri/marker) — a high-speed document-to-markdown converter supporting PDF, PPTX, DOCX, XLSX, HTML, EPUB, and images.

### Key Facts

- **What**: Submodule clone of the marker repo, used for converting documents to Markdown/JSON
- **Stack**: Python (Poetry), GPU/CPU/MPS, optional LLM boost (Gemini/Ollama)
- **Version**: 1.10.2
- **Local folder**: `20_Project/Marker/` (submodule — full repo with dependencies)

### Why Submodule

Full Python project with Poetry lockfile, test data, and benchmark assets — too large for the vault; tracked as submodule pointing to upstream fork.
