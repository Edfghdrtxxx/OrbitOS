# Fundamental Knowledge

## Context

**Objective:** Build a broad, solid foundation in physics, mathematics, and statistics — both for long-term competency and to prepare for PhD applications (target: UTokyo / RIKEN). ^ecade0

**Success Metrics:**
- [ ] Can confidently explain and solve problems across core physics (CM, EM, QM, StatMech, Nuclear)
- [ ] Comfortable with mathematical tools for both physics and ML (linear algebra, calculus, probability)
- [ ] Ready for PhD entrance exams / research interviews at UTokyo level
- [ ] Can discuss research topics fluently in technical depth

**Key Constraints:**
- Timeline: Build steadily through master's period; intensify before PhD application (late 2026)
- Resources: Textbooks, MIT OCW, 3Blue1Brown, problem sets, lecture notes — mix of all formats
- Dependencies: Directly supports [[Masters_Thesis]], [[Automated Framework for MATE(TPC)]], and PhD application

---

## Actions

### Phase 1: Mathematics Foundations
- [ ] **Linear Algebra:** Eigenvalues, SVD, matrix decompositions, vector spaces (3Blue1Brown + Strang)
- [ ] **Calculus:** Multivariable calculus, optimization, gradient methods (MIT OCW 18.02)
- [ ] **Differential Equations:** ODEs and PDEs relevant to physics (Schrödinger, heat, wave equations)
- [ ] **Probability & Statistics:** Distributions, Bayesian inference, hypothesis testing, confidence intervals

### Phase 2: Physics Core
- [ ] **Classical Mechanics:** Lagrangian/Hamiltonian formalism (Goldstein or Taylor)
- [ ] **Electromagnetism:** Maxwell's equations, wave propagation (Griffiths)
- [ ] **Quantum Mechanics:** Schrödinger equation, perturbation theory, scattering (Griffiths → Sakurai)
- [ ] **Statistical Mechanics:** Ensembles, partition functions, phase transitions (Pathria or Schroeder)
- [ ] **Nuclear/Particle Physics:** Cross-sections, kinematics, detector physics, TPC principles

### Phase 3: Statistics & Data Analysis
- [ ] **Experimental Methods:** Error propagation, fitting, chi-squared, likelihood methods
- [ ] **ML Foundations:** Loss functions, regularization, bias-variance, gradient descent theory
- [ ] **Bayesian Methods:** Prior/posterior, MCMC basics, model comparison
- [ ] **ROOT / Python Analysis:** Practical data analysis skills for HEP/nuclear experiments

### Phase 4: PhD Application Readiness
- [ ] Research UTokyo/RIKEN entrance exam format and past problems
- [ ] Solve past entrance exam problems systematically
- [ ] Prepare research statement material — connect thesis work to proposed PhD research
- [ ] Practice technical interview: explain your research clearly and answer fundamental questions
- [ ] Create summary sheets for rapid review before exams/interviews

## Notes

- UTokyo Physics PhD entrance exams typically cover: classical mechanics, electromagnetism, quantum mechanics, statistical mechanics, and mathematical physics
- RIKEN positions may focus more on research fit than exam scores — prepare both tracks
- Prioritize depth in nuclear/particle physics and QM since those align with research direction
- Use Feynman technique: write notes explaining concepts simply to test understanding
- Recommended textbooks: Griffiths (QM, EM), Goldstein (CM), Pathria (StatMech), Barlow (Statistics)
