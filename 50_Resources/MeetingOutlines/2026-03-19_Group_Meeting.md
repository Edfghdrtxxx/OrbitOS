# 组会纪要 2026-03-19

本次会议主要围绕实验数据分析、薄膜力学性能研究及核物理粒子识别方法展开讨论，并明确了后续的研究方向与任务。

## 1. 实验数据分析与薄膜力学性能研究

- **电学信号分析**：讨论了不同计数率下，薄膜电学信号的上升时间和衰减时间随计数率变化的规律。分析指出，信号的非线性变化可能与网格电极的屏蔽效应有关。
- **薄膜力学性能**：通过调整气压和距离等方法，对薄膜的形变进行了稳定测试。研究发现，薄膜的厚度变化会影响其力学性能参数，后续需将厚度作为变量纳入模型进行重新分析。
- **背景知识**：介绍了核物理实验中粒子识别（PID）的重要性，包括区分弹性与非弹性散射、研究碳聚变循环以及探测核子态转变等关键物理过程。

## 2. 核物理粒子识别方法研究

- **背景介绍**：介绍了当前核物理实验中粒子识别的挑战，传统方法依赖复杂的物理模型和手动判别，而深度学习方法因其强大的特征提取能力成为研究热点。
- **物理信息神经网络（PINN）**：提出了一种将物理知识融入深度学习模型的新方法。该方法通过引入交叉注意力机制，使模型能够关注到物理上重要的区域（如径迹末端），并利用物理特征向量与CNN提取的特征进行融合，从而提升模型的准确性和鲁棒性。
- **实验数据**：展示了在模拟数据上，该物理信息神经网络模型相较于传统CNN模型，在区分氦-3和氦-4等同位素的任务上取得了显著的性能提升。

## 后续任务

1. 完成薄膜力学性能的研究，需将厚度变化作为变量纳入模型进行重新分析。
2. 调研并补充粒子识别方法的相关背景知识，特别是关于核物理实验中粒子识别的重要性。
3. 完成初步的粒子识别工作，明确具体的研究目标和方向，以便后续进行深入分析和算法优化。
4. （导师指示）增加一个分支工作：基于Lu Li的工作，在同一任务上对比深度学习方法与传统方法的性能。
