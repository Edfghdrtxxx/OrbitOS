# A Practical Note-Taking Template

```
TOPIC: ______________________

⊙ Main point 1
    — detail / example
    → effect / result

⊙ Main point 2 ★ (emphasis)
    — detail
    BUT / ≠: contrast info

⊙ Main point 3
    eg. example (→ illustrates MP3)

Prof attitude: surprised / doubtful / enthusiastic
```

# Recommended Shorthand Symbols

| Symbol | Meaning |
|--------|---------|
| ★ | emphasized / important |
| → | leads to / causes |
| ≠ | contrast / different from |
| ? | uncertain / hedged |
| eg. | example given |
| def. | definition |
| ① ② ③ | sequence / steps |
| !! | repeated / stressed |

# Guiding Principle

> Your notes are a memory cue, not a transcript. Write key nouns and relationships — not every word. Speed matters more than completeness. You can reconstruct meaning from 3 good words.
