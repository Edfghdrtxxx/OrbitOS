# Needs improvement phrasing

## Definition

An idiomatic English pattern using "needs + noun/gerund" to state that something requires improvement. It is common in feedback and fits priority questions like "needs the most improvement."

## Key Points

- "Needs improvement" is a fixed, natural phrase in feedback contexts.
- "Needs the most improvement" is idiomatic for ranking weak areas.
- Avoid "should be necessary to improve" because it mixes necessity with an action awkwardly.
- Capture good feedback patterns in [[Daily Accumulation]] to reinforce usage.

## Examples

- "Your introduction needs improvement."
- "Which section needs the most improvement?"
