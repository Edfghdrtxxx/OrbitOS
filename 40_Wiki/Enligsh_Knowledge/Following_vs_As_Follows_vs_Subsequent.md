# Following vs As Follows vs Subsequent

## Definition

Distinguishing between terms used to introduce lists or describe temporal order in professional and technical writing. "The following" and "as follows" introduce sequences, while "subsequent" describes events occurring after a reference point.

## Key Points

- **The following**: Used as a noun or adjective; must agree in number with the linking verb (e.g., "The following *is*..." vs. "The following *are*...").
- **As follows**: A fixed idiomatic phrase that *never* changes. Incorrect forms include "as follow" or "as following."
- **Subsequent**: An adjective meaning "coming after something in time." Usually requires a reference point using "to" (e.g., "subsequent to the data collection").
- **Verb Agreement**: "As follows" requires a linking verb (is/are) before it when used to introduce a result (e.g., "The results *are* as follows").

## Examples

- "The **following are** the required dependencies for the project."
- "The steps to reproduce the bug are **as follows**:"
- "The initial experiment failed, but a **subsequent** update fixed the issue."
- "The analysis was completed **subsequent to** the data collection phase."
