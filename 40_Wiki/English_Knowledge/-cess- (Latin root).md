# -cess- (Latin root)

## Definition

**-cess-** is a Latin morpheme from the verb *cēdere* ("to go, to yield"), past participle *cessus*. It traces back to the Proto-Indo-European root \**ked-* ("to go, to yield"). Combined with different Latin prefixes, it produces a large family of English words whose meanings all derive from variants of "going" or "yielding." The standalone English word "cess" (a tax) is a false friend — it descends from a different Latin verb entirely.

## Etymology & Semantic Tree

### The Root Chain

PIE \**ked-* ("to go, to yield") → Latin *cēdere* (to go, to yield) → past participle *cessus* → English derivatives via Old French

### The Branching Tree

Each Latin prefix redirects the core meaning of "go / yield" in a different direction:

```
PIE *ked- ("to go, to yield")
  └─ Latin cēdere (to go, to yield) → past participle cessus
       │
       ├─ con- + cēdere = concēdere ("yield completely")
       │    → concession, concede
       │    "to give way, grant a point"
       │
       ├─ re- + cēdere = recēdere ("go back")
       │    → recession, recede, recess
       │    "to withdraw, pull back"
       │
       ├─ prō- + cēdere = prōcēdere ("go forward")
       │    → process, proceed, procedure, procession
       │    "to advance step by step"
       │
       ├─ ac- (ad-) + cēdere = accēdere ("go toward")
       │    → access, accede, accession
       │    "to approach, gain entry"
       │
       ├─ ex- + cēdere = excēdere ("go beyond")
       │    → excess, exceed
       │    "to surpass a limit"
       │
       ├─ suc- (sub-) + cēdere = succēdere ("go up after")
       │    → success, succeed, succession
       │    "to follow after, achieve an outcome"
       │
       ├─ prae- + cēdere = praecēdere ("go before")
       │    → precede, precedent, unprecedented
       │    "to come earlier in order"
       │
       ├─ dē- + cēdere = dēcēdere ("go away")
       │    → decease, deceased
       │    "to depart (from life)"
       │
       ├─ inter- + cēdere = intercēdere ("go between")
       │    → intercede, intercession
       │    "to mediate, step in between parties"
       │
       └─ sē- + cēdere = sēcēdere ("go apart")
            → secede, secession
            "to withdraw from a union"
```

## Key Points

- The prefix determines meaning: the same root *cēdere* produces "give in" (concede), "go back" (recede), "go forward" (proceed), "go beyond" (exceed), and "go away" (decease) — all by changing only the prefix.
- Many of these words appear on the [[Academic_Word_List_AWL|Academic Word List]] (e.g., *process*, *access*, *proceed*, *significant precedent*), making this root especially productive for academic English.
- The *-cede* / *-ceed* / *-cess* spelling variants all trace to the same Latin verb; *-cede* preserves the Latin infinitive form, *-cess* preserves the past participle, and *-ceed* is an English spelling shift (proceed, succeed, exceed).

## The False Friend: "cess" (a tax)

The standalone word **cess** — meaning a tax or levy (still used in Irish English and historical British contexts, e.g., "water cess") — does **not** come from *cēdere*. It derives from a shortened form of **assess**, which traces to:

Latin *assidēre* (ad- + *sedēre*, "to sit beside") → Medieval Latin *assessāre* (to fix a tax by sitting in judgment) → Middle English *cessen* → **cess**

The root here is *sedēre* ("to sit"), not *cēdere* ("to go") — a completely different PIE origin (\**sed-*, "to sit").

## Examples

- *The government made a **concession** on tariffs.* (con- = yielding completely)
- *The economy entered a **recession**.* (re- = going back)
- *You need badge **access** to enter the lab.* (ac- = going toward)
- *The data **processing** pipeline runs overnight.* (pro- = going forward)
- *Spending in **excess** of the budget will not be approved.* (ex- = going beyond)
- *Her **success** followed years of preparation.* (suc- = going up after)
