# Shell closure

## Definition

A shell closure occurs when a nucleon shell is filled, producing extra stability and characteristic gaps in the single-particle spectrum. These closures define [[Nuclear Magic Numbers]].

## Key Points

- The [[40_Wiki/Physics_Math/Nuclear Shell Model]] explains shell closures as large energy gaps between orbitals
- Strong [[Spin-Orbit Coupling in Nuclei]] is essential to reproduce observed magic numbers
- A sudden drop in [[Two-Neutron Separation Energy]] is a common experimental signature
- Shell effects produce deviations from the smooth [[Liquid Drop Model]] trend

## Examples

- $^4\text{He}$ ($Z = N = 2$) and $^{16}\text{O}$ ($Z = N = 8$) are classic closed-shell nuclei
- $^{208}\text{Pb}$ ($Z = 82$, $N = 126$) is doubly magic and exceptionally stable
