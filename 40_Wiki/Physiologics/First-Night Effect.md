# First-Night Effect

## Definition

The first-night effect is a well-documented sleep phenomenon where one hemisphere of the brain remains in a lighter, more vigilant state during sleep in an unfamiliar environment. This is an evolutionarily conserved survival mechanism — the brain doesn't fully trust the new setting and maintains partial alertness for potential threats.

## Key Points

- One brain hemisphere shows **reduced slow-wave (deep) sleep** while the other sleeps more normally — an asymmetric vigilance pattern (not full unihemispheric sleep like marine mammals, but a milder version of the same principle)
- Manifests as longer sleep onset latency, more awakenings, earlier spontaneous waking, and reduced total sleep time
- Typically resolves within **1-3 weeks** as the brain habituates to the new environment's sounds, temperature, smells, and tactile cues
- Distinct from chronic insomnia — the sleep architecture (deep/REM/light ratios) usually remains healthy; it's the *entry and maintenance* that suffer
- Can be accelerated by making the space familiar: personal bedding, consistent routine, masking novel sounds with white noise

## Examples

- Moving into a new student dormitory and suddenly waking at 5 AM instead of 8 AM, despite no alarm and no light changes — the brain is in sentinel mode
- Sleeping poorly the first night in a hotel but normally by night 2-3
- Sleep researchers routinely discard "night one" data in studies because this effect systematically distorts results
