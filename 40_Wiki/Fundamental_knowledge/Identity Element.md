# Identity Element

## Definition

An **identity element** for a binary operation $\circ$ on a set $S$ is an element $e \in S$ such that $e \circ a = a \circ e = a$ for every $a \in S$. The identity is unique when it exists.

## Key Points

- A [[Field (mathematics)|field]] has two identity elements: the **additive identity** $0$ (where $a + 0 = a$) and the **multiplicative identity** $1$ (where $a \times 1 = a$)
- In a [[Vector_Spaces|vector space]], the zero vector $\mathbf{0}$ is the additive identity: $\mathbf{v} + \mathbf{0} = \mathbf{v}$
- The identity matrix $I$ serves as the multiplicative identity for [[Matrix Operations|matrix multiplication]]: $AI = IA = A$
- Every [[Inverse Element|inverse element]] is defined relative to an identity: $a \circ a^{-1} = e$

## Examples

- Additive identity on $\mathbb{R}$: $0$, since $7 + 0 = 7$
- Multiplicative identity on $\mathbb{R}$: $1$, since $7 \times 1 = 7$
