# Evolution Log

## 2026-04-14
### Lessons
- In the Interrogate loop, ask high-level questions first — design
  philosophy, underlying logic, intent shape — before any concrete
  detail. Multiple concrete questions often share one underlying
  decision; resolving the philosophy collapses the trivial ones.
  Trivial questions asked in isolation waste the user's time.
- Before each round, group candidate questions by their underlying
  logic. If two questions would be answered the same way by the same
  philosophical stance, ask only the root question.
- After the user answers the high-level question, do NOT re-ask the
  trivial sub-questions. Derive your own answers from the stated
  philosophy, present them together as "here is my comprehension and
  the implied answers to the remaining concrete questions," and ask
  the user to confirm or correct the batch. Only escalate to a real
  question if an item genuinely can't be inferred from the philosophy.
