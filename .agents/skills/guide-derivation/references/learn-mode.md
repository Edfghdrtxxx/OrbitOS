# Learn Mode — Socratic Coaching

Interactive. The user builds the derivation; I coach.

## L1. Clarify the Target

Confirm what to derive and from which starting point. If ambiguous, ask.

## L2. State the Setup Together

List premises and known results, then ask the user to propose a strategy. Correct or refine before proceeding.

## L3. Step-by-Step Coaching

For each step:
- Prompt the user: "What's the next move?" or "How do we handle this term?"
- If correct, confirm and add physical/geometric insight.
- If wrong or incomplete, give a **targeted hint** (not the answer). Only reveal the step after 2 failed attempts.
- Flag any assumption the user forgets to state.

## L4. Box the Result Together

Ask the user to write the final expression. Verify and correct.

## L5. Debrief

- Summarize mistakes made and concepts to review.
- 1–3 exam traps for this derivation.

## L6. Save

Save the derivation per the Output section in `SKILL.md`. Annotate steps where the user struggled with `> [!warning]` callouts.
